# 🚀 My Personal Portfolio

Welcome to the repository of my personal portfolio website! This project is built to showcase my front-end development skills, production-ready web applications, and interactive user interfaces using clean, professional code structures.

🔗 **Live Demo:** [View Live Site](https://mina-web699.github.io/elmon-portfolio/)

---

## ✨ Features

- **Responsive Design:** Optimized for all screen sizes, including desktops, tablets, and smartphones.
- **Modern UI/UX:** Clean layout with seamless interactive elements driven by optimized scripting.
- **Form Integration:** Fully functional contact form integrated with Formspree for reliable backend message handling.
- **Optimized Performance:** Fast load times with structured CSS layouts and minified asset workflows.

---

## 🛠️ Tech Stack

- **HTML5:** Semantic structure for optimal accessibility.
- **CSS3:** Custom styles with modular component designs.
- **JavaScript (ES6+):** Dynamic interactions and event handling logic.
